# AI-Powered Task Manager

## Overview

AI-powered task management application that helps users organize work, generate subtasks using AI, estimate deadlines, and track progress.

## Features

* User Registration & Login
* JWT Authentication
* Multi-User Support
* AI Task Analysis
* Automatic Priority Assignment
* Deadline Estimation
* AI Generated Subtasks
* Task Search
* Priority Filtering
* Task Status Management
* Progress Tracking
* Edit/Delete Tasks

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Axios
* React Router

### Backend

* Node.js
* Express.js

### Database

* MySQL
* Prisma ORM

### AI Integration

* OpenRouter API

## Installation

### Backend

npm install

npm start

### Frontend

cd frontend

npm install

npm run dev
